# Prodesk IT Digital Agency Landing Page
## Project Overview
A responsive, modern landing page for Prodesk IT Digital Agency built as part of Week 1 internship assignment at Prodesk IT. This project demonstrates mastery of CSS layouts, responsive design, and modern web development practices.
## Live Demo
[View Live Site](https://your-vercel-url.vercel.app/) <!-- Replace with your actual Vercel/Netlify URL -->
## Features
✅ **Responsive Design** - Fully optimized for mobile, tablet, and desktop
✅ **Sticky Navbar** - Fixed at top during scroll for improved navigation
✅ **Dark Mode Toggle** - Persistent user preference with local storage
✅ **Micro-Interactions** - Hover effects on buttons and service cards
✅ **Mobile Menu** - Collapsible navigation for smaller screens
✅ **Service Showcase** - 3 key digital agency services presented in interactive cards
✅ **Contact Form** - Functional form with feedback UI
✅ **Modern UI** - Clean, professional design with smooth transitions
## Technologies Used
- **HTML5** - Semantic markup
- **CSS3** - Raw CSS with Flexbox/Grid layouts
- **JavaScript** - Interactive functionality
- **Font Awesome** - Icons for UI elements
